Protocols

The following protocols are available globally.

  • Coordinator

    Defines a contract that every Coordinator must abide.

    Essentially, a Coordinator is responsible to orchestrate a navigation flow between different ViewControllers in order to complete a given task.

    A Coordinator can be composed of other Coordinators. In this context, they will be called SubCoordinators.

  • Module

    Defines a contract that every Module must abide.

    A Module is used to provide an initial UIViewController for the module, as well an optional UITabBarItem.

    If provided, the UITabBarItem can be used in the App’s UITabBar.
